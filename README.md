# frontend-developer-task
## Tech Stack
- React.js
- TailwindCSS
- Node.js
- Express.js
- MongoDB

## Features
- JWT Authentication
- Protected Dashboard
- Task CRUD (Create, Read, Update, Delete)
- Secure APIs

## Setup Instructions

### Backend
```bash``
cd backend
npm install
npm run dev
## Environment Variables

Create a `.env` file inside the backend folder with the following:

PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
